## Struktura oznaczeń
Nazwy folderów odpowiadają oficjalnym kodom arkuszy CKE:
`INF.04-[Numer Arkusza]-[Rok].[Miesiąc]-SG`
* **01 / 02** – Numer wersji arkusza z danej sesji
* **SG** – Oznaczenie nowej podstawy programowej
